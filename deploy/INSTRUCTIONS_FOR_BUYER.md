# Инструкция для покупателя домена stablefold.org

Привет. Нужно купить один домен и настроить 5 DNS-записей. Времени — ~7 минут.

---

## Часть 1 — Покупка ($11)

1. Открой: https://www.namecheap.com/domains/registration/results/?domain=stablefold.org
2. Должен быть ценник около $11/год — нажми **Add to Cart**
3. **Continue to Cart** → проверь что:
   - Auto-renew — на твоё усмотрение (если ВКЛ — не забудешь продлить)
   - Domain Privacy (WhoisGuard) — **бесплатно у Namecheap, оставь ВКЛ**
4. **Confirm Order** → войди / зарегистрируйся → оплати

---

## Часть 2 — Настройка DNS (~5 минут)

После покупки:

1. Открой https://ap.www.namecheap.com/domains/list/
2. Напротив `stablefold.org` нажми **MANAGE**
3. Сверху страницы кликни вкладку **Advanced DNS**
4. В секции **HOST RECORDS** удали все имеющиеся записи (если есть Parking page / URL Redirect — удали кнопкой 🗑)
5. Нажми **+ ADD NEW RECORD** и добавь **5 записей** ровно по этой таблице:

| Type | Host | Value | TTL |
|------|------|-------|-----|
| A Record | `@` | `185.199.108.153` | Automatic |
| A Record | `@` | `185.199.109.153` | Automatic |
| A Record | `@` | `185.199.110.153` | Automatic |
| A Record | `@` | `185.199.111.153` | Automatic |
| CNAME Record | `www` | `fedorovskyfinance.github.io.` | Automatic |

⚠️ **Важно:** в CNAME-записи в конце `fedorovskyfinance.github.io` стоит **точка** — это часть имени, не пунктуация. Если Namecheap сам её добавит — ОК, не дублируй.

6. После каждой записи — зелёная галочка справа = сохранено. Если жёлтый кружок — нажми ✓ (Save).

---

## Часть 3 — Передать обратно

Напиши Максиму одно сообщение:
> Купил, DNS прописал.

Всё. Дальше Максим / его подрядчик проверяют, поднимают HTTPS, ничего больше от тебя не нужно.

---

## Если что-то пошло не так

- **«Domain is taken»** на Namecheap — напиши Максиму, поищем другой TLD
- **Не видишь Advanced DNS** — нажми **MANAGE** на домене ещё раз, должна появиться вкладка
- **Не уверен где удалить старые записи** — пришли скриншот раздела Host Records, поможем
